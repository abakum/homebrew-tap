# Abakum Tap

## How do I install on macOS?

`brew install --cask abakum/tap/crocgui`

## How do I install on Linux?

`brew install --formula abakum/tap/crocgui`
`brew install abakum/tap/crocgui`

Or `brew tap abakum/tap` and then `brew install --cask crocgui` or `brew install --formula crocgui`.

## How do I install on Windows?
`scoop bucket add abakum https://github.com/abakum/homebrew-tap`
`scoop bucket list`
`scoop install crocgui`
`scoop update crocgui`

## How do I run on macOS?

`open /Applications/crocgui.app`

## How do I run on Linux?

`crocgui`
`gtk-launch com.github.howeyc.crocgui`

## How do I run on Windows?

`crocgui`
